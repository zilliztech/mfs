# Evaluation

The evaluation directory records agent-level search and browse experiments.
It is separate from the documentation site source so large artifacts and raw
logs do not dominate the user-facing docs.

Current areas:

| Path | Scenario |
|---|---|
| `evaluation/code-search/` | Code search and verification workflows. |
| `evaluation/document-search/` | Documentation corpus search workflows. |

The public docs should summarize the benchmark shape, dataset source, and
headline interpretation. Raw corpora and very large transcripts should remain
outside the docs site unless explicitly curated.
